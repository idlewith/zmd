## 明日方舟wiki


顺便放一些我经常用的网址：
https://prts.wiki/
最经典也是最重要的网站，可以看所有干员的完整数据精二立绘，档案，方舟机制，敌人数值，剧情回顾等，只要方舟里有的prts都有
https://mapcn.ark-nights.com/
prts.map，可以看所有地图的出怪路线，种类与次序，也可以看敌人属性。这里的属性是计算过突袭/关卡加成的，更加直观
https://penguin-stats.cn/
企鹅物流数据统计，可以查看每种材料的最佳刷取地点
https://viktorlab.cn/akdata/dps/
干员伤害计算器，可以在给定条件下计算干员对敌人的输出
可惜数据只更新到涤火杰西卡版本，后面的干员和模组都没有
https://tomimi.dev/zh
方舟肉鸽（集成战略）资料库，可以看肉鸽所有关卡从0-15难度的敌人种类和属性
